# CANVAS POUTOUX GENERATOR 3.0

Generate 500 random Poutou pictures into an HTML canvas.

![pout](poutoux-demo.jpeg)

## dev 
Install deps
```
pnpm i
``` 

Serve
```
pnpm dev
```

## saving for sharing img result
*Right-click* for saving generated image.
